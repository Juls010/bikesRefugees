## Purpose

Bikes for Refugees is a web project built around a real and growing solidarity movement that has gained significant momentum across Europe. The initiative collects and donates bicycles to refugees, giving them a practical and dignified means of transport as they rebuild their lives in new communities.

The goal of the site is to raise awareness about this cause, connect people who want to help, and show the human impact behind every donated bike — all through a clear and accessible online presence.

## Approach

The project consists of a single main page (`index.html`) built with plain HTML and CSS. The approach focused on:

- Writing clean, semantic HTML to structure the content meaningfully.
- Using CSS to create a visually consistent and organized layout that communicates the mission of the initiative effectively.
- Keeping the design simple and focused on the message.

## Challenges

The main challenge during development was working with CSS layout techniques such as **Flexbox** and **Grid**. Deciding when to use each one, understanding how elements behave inside their containers, and getting the layout to look as intended required research and iteration. This challenge helped reinforce a better understanding of how CSS handles spacing, alignment, and the overall flow of a page.
